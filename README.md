# 後端 Hapi 框架

專案初始化
```
npm init
```

安裝 hapi
```
npm install @hapi/hapi
```

安裝環境變數套件，讀取 .env 檔
```
npm install dotenv
```

安裝 mongodb 套件
```
npm install mongoose
```

安裝 supervisor 偵測 js 變更自動重啟

```
npm install supervisor -g
```
